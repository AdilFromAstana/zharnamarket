# Zharnamarket PARTNER — обзор проекта

## Суть продукта
MVP-доска объявлений для Казахстана, соединяющая **бизнес** и **креаторов контента** (TikTok / Reels / Shorts) напрямую. Платформа НЕ участвует в сделке — только сводит стороны, дальше общение через Telegram / WhatsApp / телефон. Аналогия: **OLX для рекламных коллабораций с блогерами**.

## Целевая аудитория
- **Рекламодатели:** МСБ Казахстана, локальные бренды, кафе, магазины, онлайн-сервисы — те, кому дорого агентство и нужна разовая интеграция
- **Креаторы:** блогеры TikTok / Instagram Reels / YouTube Shorts (1k–1M+ подписчиков), хотят монетизировать аудиторию без агентств
- **Ролевой модели НЕТ:** один аккаунт может одновременно быть заказчиком, креатором, писать отзывы — это осознанный UX-выбор

## Актуальная модель монетизации

### Для рекламодателя (бизнеса) — платит всё
| Действие | Цена | Срок |
|---|---|---|
| **Публикация объявления** | **990 ₸** | 7 дней (обязательно) |
| Boost Rise (поднятие) | 1 990 ₸ | 7 дней |
| Boost VIP (выделение + бейдж) | 2 990 ₸ | 7 дней |
| Boost Premium (топ + выделение + бейдж) | 3 990 ₸ | 7 дней |

### Для креатора — вход БЕСПЛАТНЫЙ
| Действие | Цена |
|---|---|
| **Публикация профиля в каталоге** | **0 ₸** (бесплатно) |
| Требования | аватар + минимум 1 работа в портфолио |
| Boost Rise профиля | 1 990 ₸ / 7 дней |
| Boost VIP профиля | 2 990 ₸ / 7 дней |
| Boost Premium профиля | 3 990 ₸ / 7 дней |

**Источник истины:** [app/api/creators/[id]/publish/route.ts](../app/api/creators/[id]/publish/route.ts) — просто ставит `isPublished: true` без платежа. Константа `CREATOR_PUBLICATION_PRICE = 990` в [lib/constants.ts:266](../lib/constants.ts#L266) — легаси, не используется в актуальном flow.

### Логика модели
- Бизнес приходит с деньгами → платит за доступ к аудитории
- Креатор = сама ценность платформы → его нельзя отпугивать платой на входе
- Буст профиля — upsell для тех, кто уже увидел отдачу и хочет больше заявок

### Доп. монетизация
- **Промокоды** (% или фикс, лимит использований, срок, привязка к типу платежа)
- **Эскроу 5% комиссия** — Фаза 2+, код заложен, в MVP не используется

### Что НЕ монетизируется в MVP (сознательно)
Нет комиссии с прямых сделок, нет подписок, нет CPV/CPC/CPA — низкий порог → больше объявлений → больше платежей за публикацию и буст.

## Как зарабатывают пользователи

### Основной сценарий (≈90%) — прямой контакт
1. Креатор бесплатно публикует профиль ИЛИ просто смотрит ленту
2. Видит объявление с бюджетом (5k–500k ₸)
3. Жмёт «Связаться» → получает контакты
4. Договаривается в Telegram/WhatsApp, снимает видео
5. **Получает деньги напрямую от бизнеса** (Kaspi/карта/наличные) — платформа в расчёте не участвует

### Эскроу-сценарий (≈10%, Фаза 2+)
Бизнес замораживает бюджет, задаёт RPM (₸ за 1000 просмотров), креатор прикрепляет видео + скриншот статистики, заказчик одобряет → выплата = `(approvedViews × RPM / 1000) × 0.95` в кошелёк, 5% платформе. Вывод от 500 ₸ через Kaspi/Halyk/карту, обработка 3 рабочих дня.

## Функционал

### Для рекламодателя
- Wizard создания объявления (5 шагов, автосохранение в localStorage): название, платформа, город, категория, фото (до 5), бюджет, контакты
- 4 типа бюджета: `fixed` / `per_views` / `revenue` / `negotiable`
- Контакты: Telegram / WhatsApp / телефон / email
- Жизненный цикл: `draft → pending_payment → active → expired / paused / archived`
- Авто-истечение через 7 дней (cron)
- Статистика: просмотры + клики «Связаться»

### Для креатора
- Профиль-портфолио: соцсети с ручками и подписчиками, категории, примеры работ, скриншоты статистики
- Прайс-лист (минимальная ставка + позиции по форматам)
- Доступность: свободен / частично / занят
- Фильтры каталога: платформа, город, категория, тип бюджета, формат видео, формат интеграции, тип товара
- Отзывы 1–5 звёзд с возможностью ответа

### Справочники (DB-driven, админка)
Платформы, города, категории контента, форматы видео, форматы интеграции, типы товара, типы бюджета.

## Ключевые сущности
- **User** — email/phone, email-верификация (6-значный OTP), Google OAuth, роли user/admin, блокировка, soft-delete
- **Ad** — объявление (статусы, бюджет, контакты, бусты, viewCount, contactClickCount)
- **CreatorProfile** — профиль (портфолио, платформы, прайс, рейтинг, бусты, доступность)
- **PaymentSession** — типы: `ad_publication`, `ad_boost`, `creator_boost`, `escrow_deposit`, `wallet_topup`
- **AdBoost / CreatorBoost** — активные бусты с `expiresAt` (+7 дней)
- **EscrowAccount** — initial / spent / reserved / available
- **TaskApplication + VideoSubmission** — заявка креатора + загруженное видео
- **CreatorBalance + BalanceTransaction** — кошелёк и движения
- **WithdrawalRequest** — мин. 500 ₸, Kaspi/Halyk/карта, SLA 24ч, обработка 3 дня
- **Review** — 1–5 звёзд, возможность ответа
- **PromoCode + PromoCodeUsage** — промокоды и история

## Техническая архитектура
| Слой | Технология |
|---|---|
| Фреймворк | Next.js 16 (App Router) + TypeScript 5 |
| БД | PostgreSQL + Prisma 7 (~1064 строк схемы) |
| UI | React 19 + Ant Design 5 + Tailwind CSS 4 |
| Auth | Custom JWT (jose, HS256) + httpOnly cookies + Google OAuth + email OTP |
| Email | Nodemailer (SMTP) |
| Платежи | Абстракция `PaymentProvider` (сейчас mock, готово к Kaspi/Halyk/PayBox) |
| Аналитика | PostHog |
| Тесты | Vitest 4 — 137 интеграционных на реальной БД |

### Cron (каждые 15 мин)
- Авто-expire объявлений (7 дней)
- Поднятие по бустам: Premium 1ч, VIP 3ч, Rise 6ч
- Таймауты загрузки видео (48ч) и модерации (72ч, auto-approve)

### Безопасность
- bcryptjs для паролей
- httpOnly + Secure + SameSite=Strict cookies, access 15 мин + refresh 30 дней
- Edge middleware [proxy.ts](../proxy.ts) защищает `/admin/*`, `/cabinet/*`, `/ads/manage`, `/creators/manage`
- Дедупликация просмотров по IP-hash, фильтр ботов по User-Agent
- Защита от race condition на промокодах в транзакциях

## Roadmap
- **Фаза 1 (MVP, готово):** регистрация + верификация, публикация объявлений, каталог креаторов, оплата, промокоды, бусты, прямой контакт, отзывы, базовая админка
- **Фаза 2:** реальные Kaspi/Halyk, полная админка (модерация, блокировка, жалобы), верификация креаторов, облачное хранилище (S3/R2), FTS-поиск
- **Фаза 3:** Узбекистан/Кыргызстан, встроенный мессенджер, Telegram Login / SMS, AI-матчинг, аналитический дашборд

## KPI
Выручка, ARPU, кол-во платных публикаций/мес, доля бустов (upsell), клики «Связаться», retention рекламодателей (повторная публикация через 7+ дней). **НЕ считаем:** ROI рекламодателей, продажи, просмотры видео — вне ответственности платформы.

## Коротко
Бизнес платит 990 ₸ за объявление, креатор публикует профиль бесплатно, обе стороны могут докупать бусты за 1 990 / 2 990 / 3 990 ₸. Платформа зарабатывает на публикациях и бустах, стороны — на прямых сделках. Эскроу с 5% комиссией заложен в код, но активируется в Фазе 2.
