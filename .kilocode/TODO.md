# TODO: MVP ViralAds PARTNER

## Фокус MVP

**Цель:** Платная доска объявлений + каталог креаторов + прямой контакт

**Модель:** Бизнес платит за **публикацию объявления**, получает прямые обращения от креаторов. Платформа не участвует в переговорах и расчетах.

## Требуется реализация

### P1 — Высокий приоритет (нужно для запуска)

- [ ] **Реальный платёжный провайдер** — Kaspi Pay / Halyk Bank API. Создать класс `KaspiProvider implements PaymentProvider`, добавить в фабрику. Всё остальное (webhook, UI) уже работает.
- [ ] **Автосохранение черновика формы создания креатора** — добавить localStorage draft как в ads/new (~20 строк)
- [ ] **Админ: просмотр и обработка жалоб** — страница /admin/reports, список жалоб, статусы (new/reviewed/resolved), действия (блок/удаление)
- [ ] **Текстовый поиск по каталогам** — fulltext search по title/description в объявлениях и профилях
- [ ] **Облачное хранилище файлов** — S3 / Cloudflare R2 вместо public/uploads/

### P2 — Средний приоритет (первые месяцы)

- [ ] **Админ: управление пользователями** — список, блокировка, смена роли
- [ ] **Админ: модерация контента** — одобрение/отклонение объявлений и профилей
- [ ] **Верификация креаторов** — workflow: заявка, проверка, бейдж "Проверено"
- [ ] **Email-уведомления** — реальная отправка при: новый отзыв, истечение объявления, одобрение профиля
- [ ] **Redis rate limiting** — для production с несколькими инстансами
- [ ] **Soft delete аккаунтов** — grace period для восстановления
