# 📱 ViralAds PARTNER - Документация проекта

## 🎯 О проекте

**ViralAds PARTNER** — MVP-платформа для быстрого соединения бизнеса и креаторов в Казахстане.

> **Главная идея:** Мы — платная доска объявлений с прямым контактом. Мы не управляем сотрудничеством и не арбитрируем сделки.

Текущая логика MVP:
- бизнес **платит за публикацию объявления**;
- креатор **платит за публикацию профиля / резюме в каталоге**;
- заинтересованные стороны **выходят на прямой контакт** через кнопку «Связаться»;
- дальнейшие переговоры и расчеты происходят напрямую, вне платформы.

Главный фокус сейчас — **платное размещение объявлений + платное размещение профилей + прямой контакт**.
